安装 Hexo

使用 npm 安裝 hexo (須先安裝 Node.js)

初始化Blog

本地部署

本地预览地址：http://localhost:4000/

安裝Git

创建Github仓库

進入 GitHub 新增一個倉庫 (Repositories), 倉庫名稱為{GithubName}.github.io

官網: https://git-scm.com/downloads

廖雪峰 Git 教程: https://www.liaoxuefeng.com/wiki/0013739516305929606dd18361248578c67b8067c8c017b000

安裝完成后 在右鍵菜單中選擇 Git Bash Here , 設置 user.name 和 user.email

生成SSH密鑰文件:

默認一直回車即可，会在%userprofile%\.ssh 目录中生成 id_rsa = 私钥 、id_rsa.pub = 公钥

打開 %userprofile%\.ssh\id_rsa.pub 秘钥, 并全部複製

打開 Github_Settings_keys 頁面, 新建 SHH Key , Title 隨意填寫 , Key 粘貼 id_rsa.pub 中的内容

在Git Bash 中檢測Github公鑰是否成功: ssh git@github.com。出现下面的信息，表示成功

安装 Hexo Git

在 Blog 文件夾根目錄打開 _config.yml , 查看文檔最底部寫改爲以下

部署到Github

Blog : https://{xxxxxxxxxx}.github.io/ # {xxxxxxxx} Blog倉庫名
